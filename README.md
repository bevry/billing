## Usage

Setup your data directory

```bash
cp -R ./typescript/base ./typescript/data
```

Customise the content inside your `./typescript/data` directory.

Test and compile your project with `npm start` or `yarn start`

Deploy your `www` directory statically somewhere and set it up with cloudflare.

Deploy the cloudflare worker at `./typescript/workers/worker.ts` and apply its source customisations.

Enable cloudflare access for your domain, and enable the emails of those inside your entities.

Enjoy!
